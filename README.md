对于整个流程来说，所有图片存放到元数据 yml 中的指定位置：`typora-copy-images-to:` 

或者直接使用配置 `./${filename}_assets` 来存放图片或者其他元数据

图片现在通过PicGo传到云端使用，如果云端需要更新，直接全部下载重新上传即可

对于图片添加水印来说，使用程序将其复制到空的文件夹中，保持现有文件夹结构即可，然后全部上传到云端
